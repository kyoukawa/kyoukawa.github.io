开篇废话

应人邀请(纯粹自己闲的),我来写篇物理笔记的博客

气体的性质

状态参量

温度

定义: 表示物体冷热程度的物理量

温标

摄氏温标

规定在 1atm下

冰水混合物温度为$0^{\circ} C$

沸水温度为$0^{\circ} C$

然后等分(课内知识点)

热力学温标

开(尔文) $K$

$0K = -273.15^{\circ} C$

体积

气体的体积等于容器的容积

习惯使用$L$作为单位

$mL = cm^3$

$L = dm^3$

压强

对于气体认为其处处压强$P$相等($\rho$很小)

气体的实验定律

前提: 一定质量、温度不太低、压强不太大

波义耳-马略特定律

$T$一定,

$$PV = const$$

查理定律

$V$一定,

$$\frac{P}{T} = const$$

盖-吕萨克定律

$P$一定,

$$\frac{V}{T} = const$$

理想气体状态方程

由以上三个定律整理可得,

$$\frac{PV}{T} = const$$

亿点小知识

粒子部分(首先排除化学)

电子 e(lectron)
中子 n(eutron)
质子 p(roton)

$$_A^Z X$$

其中,$Z$为质量数,$A$为质子数=原子核所带元电荷数量=电子数

$_1^1H = _1^1P$ 质子就是氢核
$_0^1n$ 中子
$\alpha$ 粒子 $2^4He$
$\beta$ 粒子 ${-1}^0e$ 即电子
光子 $_0^0\gamma$

物质的量

$$\nu = (n) = \frac{N}{N_A}$$

$$\nu 单位为 mol$$

阿伏伽德罗常数

$$N_A = 6.02\times 10^{23} mol^{-1}$$

摩尔体积

适用于固体和液体
$$V_m = V_0 N_A$$

$$V_0为单个粒子的体积$$

摩尔质量

$$\mu = mN_A$$

$$m为单个粒子的质量$$

重要结论

在标况下 $0^{\circ} C ,1 atm$

对于气体,

$$V_m = 22.4L$$

$$\frac{PV}{T} = \nu\frac{P_0V_0}{T_0}$$

令

$$R = \frac{P_0V_0}{T_0}=8.31J/mol\cdot K$$

带入得,

$$\boxed{PV = \nu RT}$$

定义分子数密度

$$n = \frac{N}{V}$$

$$P = \frac{\nu}{V}RT = \frac{N}{VN_A}RT= n\frac{R}{N_A}T$$

规定波尔兹曼常数

$$k = \frac{R}{N_A}$$

得

$$\boxed{P = nkT}$$

分压定律

对于混合的气体

各体积相同

$$P = \sum P_i$$

分子动理论

基本理论

物质由分子组成

分子 $r \approx 10^{-10}m = 1Å$

原子核 $r \approx 10^{-15}m$

热运动

一切分子永远处于无规则运动中——热运动

扩散

布朗运动:悬在液体或气体中的微小颗粒的无规则运动

布朗运动不是热运动,布朗运动由热运动造成体现了热运动

压强的微观含义

考虑一个$(c,a,b)$的长方体

设一粒子在 $x$ 轴分速度为$v_x$,并看做理想的弹性碰撞

则其将会每$T = \frac{2c}{v_x}$撞击前表面一次

有$\Delta t$时间内,共撞击次数$N = \frac{\Delta t}{T} = \frac{v_x}{2c} \Delta t$

由弹性碰撞+动量定理+牛顿第三定律(其实很显然

每次撞击施加冲量$I = 2mv_x$ m为单个粒子的质量

则

$$\bar F = N \frac{I}{\Delta t} = \frac{v_x}{2c}\cdot2mv_x = \frac{mv_x^2}{c}$$

假设数密度$n$

$$N_0 = nabc$$

$$F_总 = \sum\frac{mv_{xi}^2}{c} = \frac{m}{c}\sum v_{xi}^2 = \frac{m}{c}\bar {v_x^2}N_0 = mnab\bar {v_x^2}$$

$$P = \frac{F_总}{S} = \frac{mnab\bar {v_x^2}}{ab} = mn\bar {v_x^2}$$

由对称性显然可得,

$$\bar {v_x^2} = \bar {v_y^2} = \bar {v_z^2} = \frac{1}{3}\bar {v^2}$$

得,

$$P = mn\frac{1}{3}\bar {v^2}$$

记平均动能

$$\bar{\epsilon_k} = \frac{1}{2}m\bar{v^2}$$

则,

$$P = \frac{2}{3}n\bar{\epsilon_k}$$

温度微观含义

前面已知

$$\boxed{P = nkT}$$

与上式联立可得,

$$\frac{2}{3}\bar{\epsilon_k} = kT$$

$$\therefore \bar{\epsilon_k} = \frac{3}{2}kT$$

即温度与平均动能成正比
