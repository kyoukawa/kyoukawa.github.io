分子动理论剩余部分

方均根

$$v_{rms} = \sqrt{\bar{v^2}} = \sqrt{\frac{3kT}{m}} = \sqrt{\frac{3RT}{mN_A}} = \sqrt{\frac{3RT}{\mu}}$$

自由度与能量均分定理

自由度

分子类型 平动自由度$t$ 转动自由度$r$ 自由度$i$
单原子分子 3 0 3
刚性双原子分子 3 2 5
刚性多原子分子 3 3 6

能量均分定理

显然对于各自由度是平等的,从而得出各自由度能量相等

$$\bar{\epsilon_k} = \frac{3}{2}kT$$

$$E_x = \frac{1}{2}m\bar{v_x^2} = \frac{1}{2}kT$$

由各自由度能量相等得,

$$E = i\cdot E_x = \frac{i}{2}kT$$

理想气体概念的完善

忽略分子体积,忽略分子间的相互作用力

$\Rightarrow$无势能

$\Rightarrow$内能 = 分子热动能

一般的,

$$E = E_p + E_k$$

由上述结论,对于理想气体

$$E = E_k = N \cdot \frac{i}{2}kT = \frac{i}{2}\nu RT$$

即

$$\boxed{E = \frac{i}{2}\nu RT}$$

速率分布律

取区间$[v,v+dv]$,有$dN$个粒子的速率$v\in[v,v+dv]$

有$f(v)$使得

$$f(v)dv = \frac{dN}{N}$$

$f(v)$称作速率分布函数

$\frac{dN}{N}$即粒子速率属于该区间的概率

因而,

$$\int_0^\infty f(v)dv = \frac{\int dN}{N} = 1$$

热力学第零定律

$A,B$处于平衡态,$B,C$处于平衡态,则$A,C$处于平衡态

其中$B$其实就充当了温度计的角色

鉴定为正确的废话

热力学第一定律

改变内能的方式:

  1. 吸放热
  2. 做功

$$Q = \Delta E + W$$

其中

$Q$表示吸放热,吸热取正、放热取负,单位为$J$

$\Delta E$ 为内能变化

$W$为对外做功

内能

$$E = \frac{i}{2}\nu RT$$

做功

$$dW = F\cdot dl = Ps\cdot dl = PdV$$

$$\therefore W = \int_{v_1}^{v_2}PdV$$

吸放热

将"固"、“液”、“气"称为"相”

相变潜热$L$

物态变化中吸放热(不计方向,均为正)

热传导

$$dQ = -\kappa(\frac{dT}{dz})dsdt$$

其中

$dQ$为传热量

$\kappa$为热导率,不是$k$是$kappa:\kappa$

$\frac{dT}{dz}$为温度梯度

负号是因为热量是高温传给低温的

理想气体准稳态过程

准稳态——压缩或膨胀过程中时刻平衡

等温过程

$T$不变

$$\Delta E = 0$$

$$W = \int_{V_1}^{V_2}PdV = \int_{V_1}^{V_2}\frac{\nu RT}{V}dV = \nu RT \ln{\frac{V_2}{V_1}}$$

$$Q = \Delta E + W = \nu RT \ln{\frac{V_2}{V_1}}$$

等压过程

$$W = \int_{V_1}^{V_2}PdV = P(V_2 - V_1) = \nu R(T_2 - T_1)$$

$$\Delta E = \frac{i}{2}\nu R(T_2 - T_1)$$

$$Q = \frac{i+2}{2}\nu R(T_2 - T_1) $$

定容(等体)过程

$$W = 0$$

$$\Delta E =\frac{i}{2}P(V_2 - V_1) = \frac{i}{2}\nu R(T_2 - T_1)$$

$$Q = \Delta E + W = \frac{i}{2}P(V_2 - V_1) = \frac{i}{2}\nu R(T_2 - T_1)$$

插播小知识

热容

$$C = \frac{\Delta Q}{\Delta T}$$

比热容

$$c = \frac{C}{m} = \frac{\Delta Q}{m\Delta T}$$

摩尔热容

定义:$1mol$物质升降$1K$吸放热

等体:$c_{V,m} = \frac{1}{\nu}(\frac{dQ}{dT})_V = \frac{1}{\nu}\frac{dE}{dT} = \frac{i}{2}R$

等压:$c_{P,m} = \frac{1}{\nu}(\frac{dQ}{dT})_P = \frac{1}{\nu}\frac{i+2}{2}\nu R = \frac{i+2}{2}R$

不难得到 迈耶公式

$$\boxed{c_{P,m}-c_{V,m} = R}$$

比热比/绝热系数

$$\gamma = \frac{c_{P,m}}{c_{V,m}} = \frac{i+2}{i}$$

绝热过程

由理想气体状态方程,

$$PV = \nu RT$$

$$\boxed{PdV + VdP = \nu R dT}$$

$$\boxed{dQ = dE + PdV = 0}$$
