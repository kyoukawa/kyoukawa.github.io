应cyh的要求,来写一下如何搭建hexo博客

搭建

创建blog文件夹并初始化

在_config.yml文件中添加这段,注意repo替换成自己的github仓库的链接

创作

在blog文件夹内hexo new test,然后生成./source/_posts/test.md

编辑test.md并保存,回到主目录

测试

在主目录hexo s会建立本地的服务器,在浏览器打开链接即可

发布

hexo clean && hexo g && hexo d
