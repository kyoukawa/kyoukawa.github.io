一些定理

高斯定理

此处与静电场高斯定理加以区分，这是数学上的

\oiintA⃗⋅ds⃗=∭(∇⋅A⃗)dV\oiint\vec{A}\cdot d\vec{s} = \iiint(\nabla\cdot\vec{A})dV ∬​A⋅ds=∭(∇⋅A)dV

斯托克斯公式

∮A⃗⋅dl⃗=∬(∇×A⃗)ds⃗\oint\vec{A}\cdot d\vec{l} = \iint(\nabla\times\vec{A})d\vec{s} ∮A⋅dl=∬(∇×A)ds

应用——推导麦克斯韦方程组

由高斯定理知

ϕ=\oiintE⃗⋅ds⃗=Qε0=∭ρdVε0\phi = \oiint\vec{E}\cdot d\vec{s} = \frac{Q}{\varepsilon_0} = \frac{\iiint \rho dV}{\varepsilon_0} ϕ=∬​E⋅ds=ε0​Q​=ε0​∭ρdV​

而

\oiintE⃗⋅ds⃗=∭(∇⋅E⃗)dV∴∭(∇⋅E⃗)dV=∭ρdVε0∴∇⋅E⃗=ρε0\oiint\vec{E}\cdot d\vec{s} = \iiint(\nabla\cdot\vec{E})dV\\ \therefore \iiint(\nabla\cdot\vec{E})dV = \frac{\iiint \rho dV}{\varepsilon_0}\\ \therefore \nabla\cdot\vec{E} = \frac{\rho}{\varepsilon_0} ∬​E⋅ds=∭(∇⋅E)dV∴∭(∇⋅E)dV=ε0​∭ρdV​∴∇⋅E=ε0​ρ​

由法拉第电磁感应定律

E=−dϕdt=−ddt∬B⃗⋅ds⃗=−∬∂B⃗∂t⋅ds⃗\mathscr{E} = -\frac{d\phi}{dt} = -\frac{d}{dt}\iint\vec{B}\cdot d\vec{s} =-\iint\frac{\partial\vec{B}}{\partial t}\cdot d\vec{s} E=−dtdϕ​=−dtd​∬B⋅ds=−∬∂t∂B​⋅ds

又

E=∮E⃗⋅dl⃗\mathscr{E} = \oint\vec{E}\cdot d\vec{l} E=∮E⋅dl

由斯托克斯

E=∮E⃗⋅dl⃗=∬(∇×E⃗)⋅ds⃗\mathscr{E} = \oint\vec{E}\cdot d\vec{l} = \iint(\nabla\times\vec{E}) \cdot d\vec{s}E=∮E⋅dl=∬(∇×E)⋅ds

∬(∇×E⃗)⋅ds⃗=−∬∂B⃗∂t⋅ds⃗∴∇×E⃗=−∂B⃗∂t\iint(\nabla\times\vec{E})\cdot d\vec{s} = -\iint\frac{\partial\vec{B}}{\partial t}\cdot d\vec{s} \\ \therefore \nabla\times\vec{E} = -\frac{\partial\vec{B}}{\partial t} ∬(∇×E)⋅ds=−∬∂t∂B​⋅ds∴∇×E=−∂t∂B​

另外两式推导略去，最终可以整理得到麦克斯韦方程组。
