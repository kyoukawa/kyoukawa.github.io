二倍角公式

\cos 2\alpha = 2\cos ^2\alpha -1$$$$
\cos 2\alpha = 1-2\sin ^2\alpha$$$$
\cos 2\alpha = \cos ^2\alpha -\sin ^2\alpha $$$$
\tan 2\alpha = \frac{2\tan\alpha}{1-\tan ^2\alpha}

\sin \frac{\alpha}{2} = \pm\sqrt{\frac{1-\cos\alpha}{2}} \
\cos \frac{\alpha}{2} = \pm\sqrt{\frac{1+\cos\alpha}{2}} \
\tan \frac{\alpha}{2} = \pm\sqrt{\frac{1-\cos\alpha}{1+\cos\alpha}} \
\therefore \tan \frac{\alpha}{2} = \frac{\sin\alpha}{1+\cos\alpha} = \frac{1-\cos\alpha}{\sin\alpha}

\sin 2\alpha = \frac{2\tan\alpha}{1+\tan ^2\alpha} \
\cos 2\alpha = \frac{1-\tan ^2\alpha}{1+\tan ^2\alpha}

a\sin x+b\sin y = \sqrt{a^2+b^2}(\frac{a}{\sqrt{a^2+b^2}}\sin x+\frac{b}{\sqrt{a^2+b^2}}\cos x)

令\cos\theta = \frac{a}{\sqrt{a^2+b^2}}, \sin\theta = \frac{b}{\sqrt{a^2+b^2}}
\theta :辅助角

\therefore 原式=\sqrt{a^2+b^2}(\cos\theta\sin x+\sin\theta\cos x) \
=\sqrt{a^2+b^2}\sin (x+\theta)\in [-\sqrt{a^2+b^2},\sqrt{a^2+b^2}]
