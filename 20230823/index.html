循环过程

$P-V$图像为闭合曲线

顺时针:正循环
净功: $W = \oint PdV = S$

逆时针:逆循环
净功: $W = \oint -PdV = -S$

热机

设热机从高温热源吸热$Q_1$,低温热源放热$Q_2$

其吸的净热$Q = Q_1 - Q_2$

由热力学第一定律,

$$Q = \Delta E + W = W$$

$$\boxed{W = Q}$$

$$\eta = \frac{W_有}{W_总}=\frac{Q}{Q_1} = \frac{Q_1 - Q_2}{Q_1} = 1 - \frac{Q_2}{Q_1}$$

卡诺循环

1~2 等温膨胀
2~3 绝热膨胀
3~4 等温压缩
4~1 绝热压缩

1~2

$$Q_1 = \nu RT_1\ln{\frac{V_2}{V_1}}$$

2~3

$$T_1V_2^{\gamma-1} = T_2V_3^{\gamma-1}$$

3~4

$$Q_2 = \nu RT_2\ln{\frac{V_3}{V_4}}$$

4~1

$$T_2V_4^{\gamma-1} = T_1V_1^{\gamma-1}$$

由绝热膨胀和绝热压缩的结论得,

$$V_2V_4 = V_1V_3$$

$$\eta = 1 - \frac{Q_2}{Q_1} = 1 - \frac{T_2\ln\frac{V_3}{V_4}}{T_1\ln\frac{V_2}{V_1}} = 1 - \frac{T_2}{T_1}$$

卡诺定理

一切工作在两热源间可逆热机的效率都为

$$\eta = 1 - \frac{T_2}{T_1}$$

一切工作在两热源间不可逆热机的效率都

$$\eta < 1 - \frac{T_2}{T_1}$$

综上对一切热机

$$\eta \le 1 - \frac{T_2}{T_1}$$

致冷机

《论用“致”还是“制”字》(gls:“把小绿本否了!”

从低温热源$T_2$吸热$Q_2$,消耗$W$,向高温热源$T_1$放出$Q_1$

$$W = Q_1 - Q_2$$

制冷系数,

$$\omega = \frac{Q_2}{W} = \frac{Q_2}{Q_1 - Q_2}$$

并有卡诺制冷循环,即把卡诺循环反过来

对于卡诺制冷循环

$$\omega = \frac{T_2}{T_1 - T_2}$$

可以大于 1

热膨胀

线膨胀系数

$$l_t = l_0(1+\alpha t),\alpha = \frac{dl}{l_0dt}$$

面膨胀密度

$$S_t = S_0(1+\gamma t),\gamma = \frac{dS}{S_0dt}$$

$$\gamma = \frac{dS}{S_0dt} = \frac{2l_0dl}{l_0^2dt} = 2\frac{dl}{l_0dt} = 2\alpha$$

体膨胀密度

$$V_t = V_0(1+\beta t),\beta = \frac{dV}{V_0dt}$$

$$\beta = \frac{dV}{V_0dt} = \frac{3l_0^2dl}{l_0^3dt} = 3\frac{dl}{l_0dt} = 3\alpha$$

液体

$$V_t = V_0(1+\beta t),\beta = \frac{dV}{V_0dt}$$

$$\beta_液 >> \beta_固$$
