开篇废话

应人邀请(纯粹自己闲的),我来写篇物理笔记的博客

气体的性质

状态参量

温度

定义: 表示物体冷热程度的物理量

温标

摄氏温标

规定在 1atm下

冰水混合物温度为$0^{\circ} C$

沸水温度为$0^{\circ} C$

然后等分(课内知识点)

热力学温标

开(尔文) $K$

$0K = -273.15^{\circ} C$

体积

气体的体积等于容器的容积

习惯使用$L$作为单位

$mL = cm^3$

$L = dm^3$

压强

对于气体认为其处处压强$P$相等($\rho$很小)

气体的实验定律

前提: 一定质量、温度不太低、压强不太大

波义耳-马略特定律

$T$一定,

$$PV = const$$

查理定律

$V$一定,

$$\frac{P}{T} = const$$

盖-吕萨克定律

$P$一定,

$$\frac{V}{T} = const$$

理想气体状态方程

由以上三个定律整理可得,

$$\frac{PV}{T} = const$$

亿点小知识

粒子部分(首先排除化学)

电子 e(lectron)
中子 n(eutron)
质子 p(roton)

$$_A^Z X$$

其中,$Z$为质量数,$A$为质子数=原子核所带元电荷数量=电子数

$_1^1H = _1^1P$ 质子就是氢核
$_0^1n$ 中子
$\alpha$ 粒子 $2^4He$
$\beta$ 粒子 ${-1}^0e$ 即电子
光子 $_0^0\gamma$

物质的量

$$\nu = (n) = \frac{N}{N_A}$$

$$\nu 单位为 mol$$

阿伏伽德罗常数

$$N_A = 6.02\times 10^{23} mol^{-1}$$

摩尔体积

适用于固体和液体
$$V_m = V_0 N_A$$

$$V_0为单个粒子的体积$$

摩尔质量

$$\mu = mN_A$$

$$m为单个粒子的质量$$

重要结论

在标况下 $0^{\circ} C ,1 atm$

对于气体,

$$V_m = 22.4L$$

$$\frac{PV}{T} = \nu\frac{P_0V_0}{T_0}$$

令

$$R = \frac{P_0V_0}{T_0}=8.31J/mol\cdot K$$

带入得,

$$\boxed{PV = \nu RT}$$

定义分子数密度

$$n = \frac{N}{V}$$

$$P = \frac{\nu}{V}RT = \frac{N}{VN_A}RT= n\frac{R}{N_A}T$$

规定波尔兹曼常数

$$k = \frac{R}{N_A}$$

得

$$\boxed{P = nkT}$$

分压定律

对于混合的气体

各体积相同

$$P = \sum P_i$$

分子动理论

基本理论

物质由分子组成

分子 $r \approx 10^{-10}m = 1Å$

原子核 $r \approx 10^{-15}m$

热运动

一切分子永远处于无规则运动中——热运动

扩散

布朗运动:悬在液体或气体中的微小颗粒的无规则运动

布朗运动不是热运动,布朗运动由热运动造成体现了热运动

分子间存在相互作用力

笔记没有gls的板书好看，所以直接拍书了(

同时存在分子势能(规定无穷处为 0),其趋势与合力类似,在$r_0$取最小值

压强的微观含义

考虑一个$(c,a,b)$的长方体

设一粒子在 $x$ 轴分速度为$v_x$,并看做理想的弹性碰撞

则其将会每$T = \frac{2c}{v_x}$撞击前表面一次

有$\Delta t$时间内,共撞击次数$N = \frac{\Delta t}{T} = \frac{v_x}{2c} \Delta t$

由弹性碰撞+动量定理+牛顿第三定律(其实很显然

每次撞击施加冲量$I = 2mv_x$ m为单个粒子的质量

则

$$\bar F = N \frac{I}{\Delta t} = \frac{v_x}{2c}\cdot2mv_x = \frac{mv_x^2}{c}$$

假设数密度$n$

$$N_0 = nabc$$

$$F_总 = \sum\frac{mv_{xi}^2}{c} = \frac{m}{c}\sum v_{xi}^2 = \frac{m}{c}\bar {v_x^2}N_0 = mnab\bar {v_x^2}$$

$$P = \frac{F_总}{S} = \frac{mnab\bar {v_x^2}}{ab} = mn\bar {v_x^2}$$

由对称性显然可得,

$$\bar {v_x^2} = \bar {v_y^2} = \bar {v_z^2} = \frac{1}{3}\bar {v^2}$$

得,

$$P = mn\frac{1}{3}\bar {v^2}$$

记平均动能

$$\bar{\epsilon_k} = \frac{1}{2}m\bar{v^2}$$

则,

$$P = \frac{2}{3}n\bar{\epsilon_k}$$

温度微观含义

前面已知

$$\boxed{P = nkT}$$

与上式联立可得,

$$\frac{2}{3}\bar{\epsilon_k} = kT$$

$$\therefore \bar{\epsilon_k} = \frac{3}{2}kT$$

即温度与平均动能成正比

# 琴生不等式

1

$$
a,b,c\in R^+,\frac{a^2}{1+a^2}+\frac{b^2}{1+b^2}+\frac{c^2}{1+c^2}=1,求证abc\le \frac{\sqrt{2}}{4}
$$

$$
法1: \
令x = a^2,y = b^2,z = c^2 \
\therefore 原命题\iff xyz \le \frac{1}{8} \
\therefore \frac{x}{1+x}+\frac{y}{1+y}+\frac{z}{1+z} = 1 \
\therefore \frac{1}{1+x}+\frac{1}{1+y}+\frac{1}{1+z} = 2 \
经展开,2xyz+xy+yz+zx = 1 \ge 4\sqrt[4]{2(xyz)^3} \
\therefore xyz \le \frac{1}{8} \
$$

$$
法2: \
令x = \frac{a^2}{1+a^2},y = \frac{b^2}{1+b^2},z = \frac{c^2}{1+c^2} \
\therefore x+y+z = 1,a^2=\frac{x}{1-x},b^2=\frac{y}{1-y},c^2=\frac{z}{1-z} \
原命题 \iff a^2b^2c^2 \le \frac{1}{8} \iff \frac{x}{1-x}\cdot\frac{y}{1-y}\cdot\frac{z}{1-z} \le \frac{1}{8} \
\frac{x}{1-x}\cdot\frac{y}{1-y}\cdot\frac{z}{1-z} = \frac{x}{y+z}\cdot\frac{y}{x+z}\cdot\frac{z}{x+y} \le \frac{x}{2\sqrt{yz}}\cdot\frac{y}{2\sqrt{xz}}\cdot\frac{z}{2\sqrt{xy}} = \frac{1}{8} (均值不等式) \
$$

$$
法3:
令a = \tan\alpha,b = \tan\beta,c= \tan\gamma \
\therefore \frac{\tan^2\alpha}{1+\tan^2\alpha}+\frac{\tan^2\beta}{1+\tan^2\beta}+\frac{\tan^2\gamma}{1+\tan^2\gamma}=1 \
\therefore \sin^2\alpha+\sin^2\beta+\sin^2\gamma = 1 \
\therefore \cos^2\alpha = \sin^2\beta+\sin^2\gamma \ge 2\sin\beta\sin\gamma(均值不等式) \
同理可证,\cos^2\beta\ge2\sin\alpha\sin\gamma,\cos^2\gamma\ge2\sin\alpha\sin\beta \
\therefore \cos^2\alpha\cos^2\beta\cos^2\gamma \ge 2\sin\beta\sin\gamma\cdot2\sin\alpha\sin\gamma\cdot2\sin\alpha\sin\beta = 8\sin^2\alpha\sin^2\beta\sin^2\gamma \
\therefore \tan^2\alpha\tan^2\beta\tan^2\gamma\ge\frac{1}{8} \
$$

2

$$
a_1,a_2,\cdots,a_n>0,求证\sqrt[n]{a_1a_2\cdots a_n}\le\frac{a_1+a_2+\cdots+a_n}{n}\
设f(x) = \ln x \
\therefore f’'(x) = -x^{-2} < 0 \
\therefore f(x)上凸 \
\because f(x)单调
原不等式 \iff \frac{1}{n}(f(a_1)+f(a_2)+\cdots+f(a_n)) \le f(\frac{a_1+a_2+\cdots+a_n}{n}),即琴生不等式
$$

3

$$
a,b,c>0,求证(abc)^{\frac{a+b+c}{3}}\le a^ab^bc^c \
$$

$$
法1:
设f(x) = x\ln x \
\therefore f’(x) = 1 + \ln x ,f’'(x) = \frac{1}{x} > 0 \
\therefore f(x)下凸 \
由\ln x单调,原命题\iff \frac{a+b+c}{3}\ln(abc)\le f(a)+f(b)+f© \
由三元均值,abc\le(\frac{a+b+c}{3})^3 \
\therefore \frac{a+b+c}{3}\ln(abc) \le 3f(\frac{a+b+c}{3}) \
由琴生不等式, 3f(\frac{a+b+c}{3})\le f(a)+f(b)+f© \
$$

$$
法2:
\because a,b,c轮换对称 \
\therefore 不妨设a\le b\le c \
\because a,b,c > 0 \
原命题\iff a^{\frac{b+c-2a}{3}}\cdot b^{\frac{a+c-2b}{3}}\cdot c^{\frac{a+b-2c}{3}} \le 1 \
a^{\frac{b+c-2a}{3}}\cdot b^{\frac{a+c-2b}{3}}\cdot c^{\frac{a+b-2c}{3}} = (\frac{a}{b})^{\frac{b-a}{3}}\cdot(\frac{b}{c})^{\frac{b-c}{3}}\cdot(\frac{a}{c})^{\frac{c-a}{3}} \
\because 0\le a\le b\le c \
\therefore (\frac{a}{b})^{\frac{b-a}{3}}\cdot(\frac{b}{c})^{\frac{b-c}{3}}\cdot(\frac{a}{c})^{\frac{c-a}{3}} \le 1 \
$$

排序/切比雪夫不等式

4

$$
a,b,c\in R^+,abc=1,求证a^2+b^2+c^2\le a^3+b^3+c^3 \
不妨设a\le b\le c \
由排序不等式, \
\left{
\begin{aligned}
a^3+b^3+c^3 = a^2\cdot a+b^2\cdot b+c^2\cdot c \
a^3+b^3+c^3 \ge a^2\cdot b+b^2\cdot c+c^2\cdot a \
a^3+b^3+c^3 \ge a^2\cdot c+b^2\cdot a+c^2\cdot b \
\end{aligned}
\right. \
\therefore a^3+b^3+c^3 \ge \frac{1}{3}(a+b+c)(a^2+b^2+c^2) \ge \frac{1}{3}\cdot 3\sqrt[3]{abc}(a^2+b^2+c^2) = a^2+b^2+c^2 \
$$

赫尔德不等式Hölder

$$
a_i,b_i(1\le i\le n)\in R^+,\alpha >0,\beta >0,\alpha + \beta = 1 \
则\sum_{i=1}^n a_i^\alpha b_i ^\beta \le (\sum_{i=1}^n a_i)^\alpha\cdot (\sum_{i=1}^n b_i)^\beta \
$$

证明

$$
\sum_{i=1}^n a_i^\alpha b_i ^\beta \le (\sum_{i=1}^n a_i)^\alpha\cdot (\sum_{i=1}^n b_i)^\beta \
\iff \frac{\sum_{i=1}^n a_i^\alpha b_i ^\beta}{(\sum_{i=1}^n a_i)^\alpha\cdot (\sum_{i=1}^n b_i)^\beta} \le 1 \
\iff \sum_{i=1}^n (\frac{a_i}{\sum_{j=1}^n a_j})^\alpha (\frac{b_i}{\sum_{j=1}^n b_j})^\beta \le 1 \
\ln ((\frac{a_i}{\sum_{j=1}^n a_j})^\alpha (\frac{b_i}{\sum_{j=1}^n b_j})^\beta) = \alpha \ln\frac{a_i}{\sum_{j=1}^n a_j}+\beta\ln \frac{b_i}{\sum_{j=1}^n b_j}
\le \ln(\frac{\alpha a_i}{\sum_{j=1}^n a_j}+\frac{\beta b_i}{\sum_{j=1}^n b_j}) \
由\ln x单调性,(\frac{a_i}{\sum_{j=1}^n a_j})^\alpha (\frac{b_i}{\sum_{j=1}^n b_j})^\beta\le\frac{\alpha a_i}{\sum_{j=1}^n a_j}+\frac{\beta b_i}{\sum_{j=1}^n b_j} (琴生不等式)\
\therefore \sum_{i=1}^n (\frac{a_i}{\sum_{j=1}^n a_j})^\alpha (\frac{b_i}{\sum_{j=1}^n b_j})^\beta\le\sum_{i=1}^n \frac{\alpha a_i}{\sum_{j=1}^n a_j}+\frac{\beta b_i}{\sum_{j=1}^n b_j} = \alpha + \beta = 1 \
$$

小练习第9题解析

Posted on 2023-02-16 Edited on 2023-09-12 In 旧博客搬运

9

$$设f_k(x)为n=k时左式的值$$

$$f_k(x) = \sum_{i=0}^k \frac{x^i}{i!}$$

$$\therefore f_k’(x) = \sum_{i=0}^k \frac{i\cdot x^{i-1}}{i!} = \sum_{i=0}^{k-1} \frac{x^i}{i!} = f_{k-1}(x)$$

$$易知n = 0,1时命题成立$$

偶推奇

$$假设n=k(\frac{k}{2} \in N)时命题成立$$

$$\therefore f_k(x)没有零点$$

$$又\because \lim_{x\to +\infty}f_k(x) = +\infty$$

$$\therefore f_k(x)恒大于0$$

$$\therefore f_{k+1}'(x)恒大于0$$

$$\therefore f_{k+1}(x)单增$$

$$\lim_{x\to -\infty} f_{k+1}(x) = \lim_{x\to -\infty} 1 + x + \frac{x^2}{2!} + \cdots +\frac{x^{k+1}}{(k+1)!} = 1(1+x) + \frac{x^2}{2!}(1+\frac{x}{3}) + \cdots = -\infty$$

$$显然\lim_{x\to +\infty} f_{k+1}(x) = +\infty$$

$$\therefore f_{k+1}(x)值域为R, f_{k+1}(x)有且只有一个零点$$

$$\therefore 此时有唯一解$$

$$\therefore n = k+1时,命题成立$$

奇推偶

$$假设n=k(k = 2l+1,l \in N)时命题成立$$

$$\therefore f_k(x)在(-\infty,+\infty)上单增$$

$$\therefore f_{k+1}'(x)在(-\infty,+\infty)上单增$$

$$\therefore f_{k+1}(x)先减后增$$

$$\therefore f_{k+1}(x)存在最小值,设最小值在x_0取到,f_{k+1}'(x_0) = 0$$

$$\therefore f_k(x_0) = 0,显然x_0\neq 0$$

$$f_{k+1}(x){min} = f{k+1}(x_0) = f_k(x_0) + \frac{x^{k+1}}{(k+1)!} = \frac{x^{k+1}}{(k+1)!} > 0$$

$$\therefore\forall x\in R,f_{k+1}(x)>0, 所以方程无解$$

$$\therefore n = k + 1时命题成立$$

$$综上所述,n\ge0时,命题成立$$

$$证毕$$

立体几何

Posted on 2023-01-19 Edited on 2023-09-12 In 旧博客搬运

多面体、旋转体

  1. 多面体:由若干个平面多边形围成的几何体
  2. 旋转面:一条平面曲线(直线)绕所在平面内一条定直线(轴)旋转形成的曲面
  3. 旋转体:封闭的旋转面围成的几何体

棱柱

概念

  1. 两个面互相平行(底面)
  2. 其余各面都是四边形
  3. 相邻四边形公共边互相平行(平行四边形)

分类

  1. 底面的边数:三棱柱、四棱柱、五棱柱…
  2. 侧棱是否垂直底面
    1. 垂直——直棱柱
      1. 底面为正多边形——正棱柱(特殊的直棱柱)
    2. 不垂直——斜棱柱
      1. 平行六面体——六面都是平行四边形

棱锥

概念

  1. 一面为多边形(退化)
  2. 其余各面为三角形

分类

  1. 按底面边数分:三棱锥、四棱锥、五棱锥…
  2. 底面是正多边形,顶点与底面中心连线垂直于底面:正棱锥
    1. 侧面的三角形全等

棱台

概念

  1. 用一个平行于底面的平面去截棱锥,底面与截面之间那部分——棱台

分类

由n棱锥截成n棱台

圆柱、圆锥、圆台

圆柱

坑:圆柱侧面不是矩形是曲面,所以不能用定义棱柱的方法定义

圆柱:矩形以矩形的一边所在直线为旋转轴,其余三边旋转一周形成的旋转体

表示:圆柱$O O^{'}$

圆锥

以直角三角形的一条直角边所在直线为旋转轴,旋转一周形成的旋转体

表示:圆锥$SO$

圆台

用平行于圆锥底面的平面去截圆锥,底面与截面之间的部分——圆台

表示:圆台$OO^{'}$

球

半圆以它直径所在直线为旋转轴,旋转一周形成的曲面叫做球面(半圆只有一段弧)

球面围成的旋转体叫做球体,简称球

表示:球$O$

简单组合体

概念

由简单几何体组合形成的几何体叫做简单组合体

两种构成方式

  1. 由简单几何体拼接而成
  2. 由简单几何体截取或挖去一部分而成

直观图

画法

常用斜二画图法

步骤

画轴

$\angle x^{‘}o^{’}y^{'} = 45^{\circ}$ 或 $135^{\circ}$

画线

  • 平行$x$轴的画为平行$x^{'}$
  • 平行$y$轴的画为平行$y^{'}$
  • 其他方向分解成向量处理

取长度

  • 平行$x$轴的线段长度不变
  • 平行$y$轴线段长度为原来的一半
  • 其他方向分解成向量处理

补充

三变与三不变

  • 三变
    • 坐标系夹角
    • $y$轴方向长度
    • 形状
  • 三不变
    • 平行性
    • $x,z$轴方向长度
    • 相对位置

面积(与$x,y$轴所在平面平行)

$S^{'} = \frac{\sqrt{2}}{4}S$
